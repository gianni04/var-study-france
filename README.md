# VaR Study France

Cours complet et projet pratique sur la **Value-at-Risk (VaR)** appliquée à trois classes d'actifs françaises :
- **Action** : TotalEnergies (`TTE.PA`)
- **Indice** : CAC 40 (`^FCHI`)
- **Obligation** : OAT France 10 ans (proxy taux/oblig obligataire)

## Objectif
Construire une étude complète, pédagogique et quantitative de la VaR :
- rendements et risque
- VaR historique
- VaR paramétrique
- VaR Monte Carlo
- Expected Shortfall / CVaR
- backtesting de Kupiec
- comparaison entre actifs
- interprétation économique

## Structure prévue
- `README.md` : présentation du projet
- `cours_var_france.md` : cours détaillé
- `main.py` : script principal
- `requirements.txt` : dépendances
- `output/` : graphiques et résultats
